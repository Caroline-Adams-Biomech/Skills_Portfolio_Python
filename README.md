### Python Skills
The purpose of this repository is to outline the development of technical skills with python. Initally to get to grips with the coding language I completed two online courses, which gave me a feel for Python but was in the DataCamp browser and wasn't specific to biomechanics. So I then created this jupyter notebook, which demonstrates my progress working through the kineticstoolkit tutorials. https://kineticstoolkit.uqam.ca/doc/getting_started_intro.html
Along the way I reflected on my coding issues and went from novice to competent user able to comprehend a range of data structures, use different packages, modules and methods within the python programming environment to import biomechanics data and proces it, as can be seen by scrolling through the notebook.
